# MedGrid — Hospital Management System

MedGrid is a hospital management web application being built to demonstrate a production-grade DevOps pipeline on AWS.

The application handles core hospital workflows — patient registration, appointment scheduling, doctor assignment, billing, and medical records. It is built as containerised microservices and deployed to AWS EKS through a fully automated CI/CD pipeline.

---

## Project Focus

The primary focus of this project is **DevOps engineering**, not application features.

The goal is to design, implement, and maintain a complete CI/CD pipeline that automates the build, test, and deployment process for a real-world containerised application — using AWS managed services including CodePipeline, CodeBuild, CodeDeploy, CodeCommit, ECR, and EKS, alongside GitHub.

---

## What Will Be Built

- A CI/CD pipeline that triggers automatically on every code push
- Docker containerisation of each microservice
- Automated code quality checks and security scanning
- Image build and push to Amazon ECR
- Deployment to Amazon EKS with three strategies — Blue-Green, Rolling, and Canary
- Infrastructure provisioned entirely as code using Terraform
